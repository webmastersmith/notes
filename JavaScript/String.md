# String

## Iterate

- for of

```js
for (const c of str) {
  console.log(c);
}
```

- spread

```js
const s = [...str].map((c) => c);
```

## [Methods](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)

**[includes](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/includes)**

```js
str.includes("s");
```
